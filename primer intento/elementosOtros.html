Elementos y Características "Otros" en HTML5

Categoría Elemento/Característica Descripción
Incrustación de Contenido <object> Se utiliza para incrustar objetos externos, como archivos multimedia, dentro de una página web.
<iframe> Se utiliza para incrustar contenido web externo, como páginas de otros sitios, dentro de una página web.
<embed> Se utiliza para incrustar contenido multimedia, como audio o video, en una página web.
<applet> Se utilizaba en HTML4 para incrustar applets de Java, pero es obsoleto en HTML5.
<map> Define un mapa de imágenes para vincular áreas de una imagen a diferentes destinos.
<area> Define una zona en un mapa de imágenes creado con <map>.
Compatibilidad con JavaScript Deshabilitado <noscript> Proporciona contenido alternativo que se muestra si JavaScript está deshabilitado en el navegador del usuario.
<canvas> Proporciona un lienzo en el que se pueden realizar dibujos de gráficos rasterizados a través de JavaScript.
<bdo> Define la dirección de texto para controlar la escritura bidireccional en páginas web.
Otras Características <details> Se utiliza para crear áreas colapsables en una página web que muestran u ocultan contenido adicional.
<mark> Resalta partes de un texto para destacarlas visualmente.
<abbr> Define una abreviatura o acrónimo y proporciona una descripción completa de la abreviatura.
<figure> Define una figura o ilustración con una descripción opcional en una página web.
<figcaption> Proporciona una leyenda para una figura definida con <figure>.
<time> Define una fecha o hora y puede incluir máquinas para la interpretación de fechas por parte de los navegadores.
