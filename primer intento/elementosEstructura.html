Tabla de Elementos de Estructura en HTML5

Categoría Etiqueta Descripción
Elementos Raíz <html> Define el inicio y el final de un documento HTML.
<head> Contiene metadatos, enlaces y scripts del documento.
<body> Contiene el contenido visible de la página.
Elementos de Metadatos <title> Define el título visible de la página en la barra de título del navegador.
<meta> Proporciona metainformación, como el juego de caracteres, sobre el documento.
<link> Vincula el documento a hojas de estilo externas o iconos.
<script> Incluye scripts, generalmente JavaScript.
Elementos de Encabezado <header> Define la cabecera de la página, a menudo contiene logotipos y enlaces de navegación.
<footer> Define el pie de página de la página, suele incluir información de contacto o derechos de autor.
<nav> Contiene enlaces de navegación, como menús de navegación.
<section> Define una sección temática de la página. Puede contener encabezados y contenido relacionado.
<article> Define un artículo independiente y autocontenido, como una publicación de blog.
<aside> Contiene contenido relacionado pero no esencial para el artículo principal.
Elementos de Agrupación <div> Agrupa y estructura el contenido sin ningún significado semántico específico.
<span> Agrupa elementos en línea sin cambiar el flujo del texto.
<main> Define el contenido principal de la página, generalmente un único elemento en la página.
<figure> Agrupa contenido, como imágenes, con sus descripciones (<figcaption>).
Elementos de Texto <p> Define un párrafo.
<h1>, <h2>, ... <h6> Encabezados de diferentes niveles, desde el más alto (h1) al más bajo (h6).
<ul>, <ol> Crea listas desordenadas y ordenadas, respectivamente.
<li> Elementos de lista dentro de <ul> o <ol>.
<blockquote> Cita textual o bloque de citas.
Elementos de Formulario <form> Define un formulario para la entrada de datos.
<input> Elemento de entrada en un formulario, con varios tipos como texto, contraseña, etc.
<button> Botón interactivo en un formulario.
<select> Crea listas desplegables para seleccionar opciones.
<textarea> Área de texto de múltiples líneas en un formulario.
Elementos Multimedia <img> Inserta imágenes en la página.
<audio> Incrusta contenido de audio, con controles de reproducción.
<video> Incrusta contenido de video, con controles de reproducción.
<object> Incrusta objetos multimedia, como contenido Flash (menos común en la actualidad).
Elementos de Tabla <table> Crea tablas en la página.
<tr> Define filas en una tabla.
<th> Define encabezados de celda en una tabla, generalmente en la primera fila.
<td> Define celdas de datos en una tabla.
