Tabla de Elementos de Contenido en HTML5

Categoría Elementos Descripción
Texto <p>, <h1> - <h6>, <span>, <br> Elementos utilizados para mostrar y estructurar texto en la página web.
Multimedia <audio>, <video>, <img> (considerado en esta categoría para la incorporación de imágenes relacionadas con el contenido multimedia) Elementos que permiten la incorporación y reproducción de contenido multimedia, como audio, video e imágenes relacionadas con contenido multimedia.
Incrustaciones <iframe>, <object>, <embed> Elementos utilizados para incrustar contenido de otras fuentes en una página web.
Formularios <form>, <input>, <button>, <select>, <textarea> Elementos que se utilizan para crear formularios interactivos que permiten a los usuarios enviar datos.
Contenido Interactivo <canvas>, <svg>, <math> Elementos que se utilizan para crear contenido interactivo en la página, como juegos y gráficos.
Contenido Reutilizable <template> Elemento que se utiliza para definir contenido reutilizable en diferentes partes de la página.
Marcas Semánticas y Estructura <header>, <nav>, <main>, <article>, <section>, <aside>, <footer> Elementos que proporcionan significado y estructura al contenido de la página, ayudando a organizar el contenido de manera semántica.
Grupos de Contenido <div>, <span>, <details> Elementos utilizados para agrupar y estructurar contenido de manera genérica.
Contenido de Texto Inline <a>, <em>, <strong>, <abbr>, <code>, <mark>, <cite>, <dfn>, <q>, <blockquote>, <del>, <ins> Elementos que se utilizan para aplicar estilos y efectos al texto dentro de un párrafo u otro elemento de texto.
