API de Arrastrar y Soltar en HTML5

Categoría Elemento/Característica Descripción
Elementos Principales Elementos Arrastrables Elementos que se pueden arrastrar, como imágenes, enlaces, etc.
Destinos de Soltar Áreas en las que se pueden soltar los elementos arrastrados.
Ghost Image Una imagen fantasma que representa el elemento arrastrado durante la operación.
Eventos de Arrastrar y Soltar dragstart Se dispara cuando comienza el arrastre de un elemento.
dragenter Se dispara cuando un elemento se arrastra sobre un objetivo de soltar.
dragover Se dispara mientras se arrastra un elemento sobre un objetivo de soltar.
dragleave Se dispara cuando un elemento sale de un objetivo de soltar.
drop Se dispara cuando se suelta un elemento en un objetivo de soltar.
dragend Se dispara cuando se completa la operación de arrastrar y soltar.
Atributos y Propiedades draggable Atributo que indica si un elemento es arrastrable o no.
dataTransfer Propiedad que permite transferir datos durante la operación de arrastrar y soltar.
Efectos y Feedback Feedback Visual Cambios visuales en el elemento o destino durante el arrastre para proporcionar retroalimentación al usuario.
effectAllowed Propiedad que controla el efecto de arrastre permitido, como "mover" o "copiar".
API de Datos DataTransfer API Proporciona métodos para transferir datos durante la operación de arrastrar y soltar.
setDragImage() Método que permite establecer una imagen personalizada como representación del elemento arrastrado.
clearData() Método que borra los datos asociados con la operación de arrastrar y soltar.
Restricciones y Comportamiento Personalizado dropEffect Propiedad que controla el efecto de soltar predeterminado.
allowedEffects Propiedad que especifica los efectos de soltar permitidos.
Personalización y Estilos Estilos CSS Se pueden aplicar estilos personalizados para indicar elementos arrastrables y áreas de destino.
Custom Feedback Personalización de la retroalimentación visual durante la operación de arrastrar y soltar.
Operaciones Avanzadas Arrastrar entre Ventanas/Documentos Permite arrastrar elementos entre ventanas o documentos web diferentes.
Arrastrar y Soltar Archivos Permite arrastrar y soltar archivos desde el sistema de archivos del usuario.
