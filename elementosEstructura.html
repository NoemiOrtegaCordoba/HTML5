Etiquetas de estructura

Categoría Etiqueta Descripción
Encabezado <header> Crea un encabezado
<footer> Crea un pie de página
<nav> Crea una barra de navegación
<aside> Crea un contenido secundario
Organización <div> Crea un contenedor para agrupar elementos HTML
<section> Crea una sección de contenido
<article> Crea un artículo de contenido
<main> Define el contenido principal de la página web
Listas <ul> Crea una lista no numerada
<ol> Crea una lista numerada
<li> Crea un elemento de una lista
Cuadros <table> Crea una tabla
<tr> Crea una fila de una tabla
<th> Crea una celda de encabezado de una tabla
<td> Crea una celda de datos de una tabla
Formularios <form> Crea un formulario
<input> Crea un campo de entrada
<textarea> Crea un área de texto
<select> Crea un menú desplegable
<option> Crea una opción en un menú desplegable
<button> Crea un botón para iniciar una acción.
<label> Crea una etiqueta para un campo de entrada
