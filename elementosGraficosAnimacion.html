Elementos de Gráficos y Animación en HTML5

Categoría Elemento/Característica Descripción
Gráficos Vectoriales <svg> Define gráficos vectoriales escalables.
<circle> Define un círculo en un gráfico SVG.
<rect> Define un rectángulo en un gráfico SVG.
<line> Define una línea en un gráfico SVG.
<path> Define un camino o forma personalizada en un gráfico SVG.
<polygon> Define un polígono en un gráfico SVG.
<text> Permite la inclusión de texto en gráficos SVG.
Gráficos Raster <canvas> Proporciona un lienzo en el que se pueden realizar dibujos de gráficos rasterizados a través de JavaScript.
Animación CSS Transitions Permite animar cambios en propiedades CSS como color, tamaño y posición.
CSS Animations Permite crear animaciones CSS personalizadas con keyframes.
<animate> Define animaciones en gráficos SVG.
<animateTransform> Define animaciones de transformación en gráficos SVG.
<animateMotion> Define animaciones de movimiento en gráficos SVG.
Gráficos 3D WebGL Proporciona una API para crear gráficos 3D interactivos en el navegador.
Librerías y Frameworks Three.js Una librería para crear gráficos 3D en la web.
