Elementos de Formulario Avanzados en HTML5

Categoría Elemento Descripción
Entrada de Datos <input> Elemento de entrada en un formulario, con tipos avanzados como fecha, hora, teléfono, etc.
<textarea> Área de texto de múltiples líneas en un formulario.
<select> Define un menú desplegable para seleccionar opciones.
<datalist> Define una lista de opciones para su uso con el elemento <input> tipo "text".
<input type="file"> Permite a los usuarios cargar archivos desde su dispositivo.
<input type="checkbox"> Permite a los usuarios seleccionar múltiples opciones.
Controles de Formulario Personalizados <input type="range"> Permite a los usuarios seleccionar un valor dentro de un rango.
<input type="color"> Permite a los usuarios seleccionar un color.
<input type="range"> Permite a los usuarios seleccionar un valor dentro de un rango.
Validación de Formulario <input type="email"> Verifica que la entrada sea una dirección de correo electrónico válida.
<input type="url"> Verifica que la entrada sea una URL válida.
<input type="tel"> Verifica que la entrada sea un número de teléfono válido.
<input type="number"> Permite la entrada de números y realiza validación numérica.
Selección de Fechas y Tiempo <input type="date"> Permite a los usuarios seleccionar una fecha.
<input type="time"> Permite a los usuarios seleccionar una hora.
<input type="datetime-local"> Permite a los usuarios seleccionar fecha y hora local.
<input type="week"> Permite a los usuarios seleccionar una semana.
<input type="month"> Permite a los usuarios seleccionar un mes.
Controles de Rango <input type="range"> Permite a los usuarios seleccionar un valor dentro de un rango.
Contenido Multimedia en Formularios <input type="file"> Permite a los usuarios cargar archivos multimedia.
Elementos de Salida de Formulario <output> Se utiliza para mostrar el resultado de cálculos realizados en el formulario.
Formularios Ocultos <input type="hidden"> Se utiliza para almacenar datos ocultos en el formulario que se envían al servidor cuando se envía el formulario.
Formularios Interactivos <button> Elemento botón interactivo que puede activar eventos.
<fieldset> Define un grupo de controles de formulario relacionados.
<legend> Define un título para el elemento <fieldset>.
<label> Etiqueta de texto para un elemento de formulario.
<input type="button"> Un botón de formulario estándar que no realiza ninguna acción por sí mismo.
