Elementos de Semántica y Accesibilidad en HTML5

Categoría Elemento/Característica Descripción
Estructura de Documento <header> Define la cabecera de un documento o sección.
<nav> Define una sección de navegación.
<main> Define el contenido principal de un documento.
<article> Define un contenido independiente y autocontenido, como una publicación de blog o un artículo de noticias.
<section> Define una sección temática dentro de un documento.
<aside> Define contenido relacionado pero no esencial.
Elementos de Texto y Títulos <h1>, <h2>, ... Definen encabezados de diferentes niveles.
<p> Define un párrafo.
Listas y Elementos de Definición <ul>, <ol> Definen listas no ordenadas y ordenadas.
<li> Define un elemento de lista.
<dl> Define una lista de definición.
Elementos de Enlace <a> Define un enlace.
<nav> Define una sección de navegación.
<menu> Define un menú de opciones.
Elementos de Formularios <form> Define un formulario.
<input> Define un campo de entrada de datos en un formulario.
<label> Etiqueta de texto para un elemento de formulario.
<button> Elemento botón interactivo para formularios.
Elementos de Contenido Incrustado <figure> Define contenido incrustado, como imágenes, videos o gráficos, junto con una descripción.
<figcaption> Define una descripción para el contenido dentro de un elemento <figure>.
Elementos de Accesibilidad <mark> Resalta texto para llamar la atención.
<strong> Indica un texto fuerte o importante.
<em> Enfatiza el texto.
<abbr> Define una abreviatura.
