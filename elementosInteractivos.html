Elementos Interactivos en HTML5

Categoría Elemento Descripción
Formularios <form> Define un formulario para la entrada de datos.
<input> Elemento de entrada en un formulario, con varios tipos como texto, contraseña, etc.
Contenido Multimedia <audio> Incrusta contenido de audio en la página.
<video> Incrusta contenido de video en la página.
Contenido Dinámico y Scripts <script> Permite la ejecución de código JavaScript para contenido dinámico.
<canvas> Define una zona gráfica para dibujos mediante JavaScript.
Botones y Enlaces <button> Elemento botón interactivo que puede activar eventos.
<a> Define un hipervínculo o enlace.
Elementos de Entrada de Usuario <select> Define un menú desplegable para seleccionar opciones.
<textarea> Área de texto de múltiples líneas en un formulario.
Contenido Emergente y Ventanas Modales <dialog> Se utiliza para mostrar contenido emergente o ventanas modales que requieren interacción del usuario.
<details> Define un elemento con contenido oculto que se puede mostrar u ocultar.
Elementos de Control de Contenido <details> Define un elemento con contenido oculto que se puede mostrar u ocultar.
<summary> Define el resumen o título visible de un elemento <details>.
Arrastrar y Soltar Eventos de arrastrar y soltar HTML5 Permiten a los usuarios arrastrar elementos y soltarlos en áreas específicas de una página web.
