Elementos Multimedia en HTML5

Categoría Elemento Descripción
Audio <audio> Permite la incorporación y reproducción de contenido de audio en una página web.
<source> Se utiliza junto con <audio> para especificar diferentes formatos de archivo de audio para una mayor compatibilidad.
Video <video> Permite la incorporación y reproducción de contenido de video en una página web.
<source> Se utiliza junto con <video> para proporcionar diferentes formatos de video para una mayor compatibilidad.
Imagen y Gráficos Vectoriales <img> Se utiliza para la incorporación de imágenes estáticas en una página web.
<svg> Define gráficos vectoriales escalables en la página.
Contenido Matemático <math> Se utiliza para representar contenido matemático en la web.
Mapas de Imagen y Áreas Interactivas <map> Se utiliza para crear imágenes con áreas interactivas que pueden contener enlaces y activar eventos.
<area> Define una zona en un mapa de imagen que se puede utilizar para enlaces y eventos interactivos.
